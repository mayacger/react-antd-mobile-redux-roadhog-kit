# create-react-app react react-router antd-mobile redux redux-saga kit

![Alt text](/demo.png)

## 本地使用

```
npm i
npm start

```
浏览器打开 http://localhost:3000/

## 生产发布 

```
npm run build

```
打包后的文件在 build 文件夹内

### 测试 build 文件
```
yarn global add serve
serve -s build

```
## 简单说明


* 使用的react-router 3X，
* react "15.4.0"，测试中发现antd-mobile和react最新版有点问题，有意想不到的bug，所以采用15.4.0。



## 这是一个demo kit，不建议正式使用；
