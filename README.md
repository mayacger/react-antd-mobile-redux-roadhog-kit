# react-antd-mobile-redux-roadhog-kit
react-antd-mobile-redux-roadhog-kit


## 说明

react antd-mobile
roadhog构建服务


参考：https://github.com/sorrycc/roadhog
roadhog 资源文件必须在src、public。默认没有说明；
### 安装

```bash
$ npm i roadhog -g
$ npm i
```

### 使用

本地开发

```bash
$ roadhog server
```

打包发布

```bash
$ roadhog build
```
